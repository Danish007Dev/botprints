How to read BotPrints

🚀 Quick Start (3 Steps)

  1. Configure Thresholds — Go to ⚙️ Settings and set your Low/Medium/High risk cutoffs. Start conservative (60/80/90).
  2. Set Appeal Message — Customize the message users see when content is removed. Use {username} and {subreddit} variables.
  3. Choose Opt-ins — Enable Raid Alerts, New Account Amplifier, and Shared Threat Layer based on your community's needs.

Suspicion Score (0-100): Composite of 6 behavioral signals. Higher = more anomalous behavior.

4-Tier Enforcement:
• 👁 Watch (Tier 0): Silent monitoring — get modmail alerts on new activity.
• 🔽 Filter (Tier 1): Future content auto-routed to modqueue for review.
• ⚠ Remove + Appeal (Tier 2): Content removed, user gets a chance to appeal via modmail.
• 🚫 Ban + Report (Tier 3): Permanent ban, content reported to Reddit admins.

Badges:
• ✓ Consistent: Normal, steady behavior.
• ⚡ Shift: Sudden change in posting patterns (possible account purchase).
• 🔗 Bot Ring: Posts at the exact same time as other accounts.
• 🆕 New Account: Account age below threshold — score amplified for extra scrutiny.
• 🕵️ Ban Evader: Behavioral fingerprint matches a previously banned account.
• 🌐 Shared Intel: Flagged by the cross-subreddit threat network.

Signals (6-Axis Radar Chart):
• Timing (Temporal regularity): Unnatural posting regularity — bots use cron jobs.
• Circadian (Circadian entropy): Posts 24/7 without a sleep cycle.
• Engagement (Engagement ratio): Abnormal post-to-comment ratio.
• Edits (Edit absence): Never edits — AI doesn't make typos.
• Bursts (Burst-silence): Long silence then batch posting spam.
• Votes (Vote correlation): Suspiciously uniform upvote patterns (astroturfing).

Advanced Features:
• ⚠️ Raid Detection: Automatic alert when multiple suspicious accounts post within minutes.
• 🤖 AutoMod Generator: Auto-generates ready-to-use AutoMod rules from detected patterns.
• 🕵️ Ban Evasion Detection: Cosine similarity matching against banned users' behavioral fingerprints.
• 🌐 Shared Threat Layer: Cross-subreddit intelligence sharing for confirmed bot rings (opt-in).
